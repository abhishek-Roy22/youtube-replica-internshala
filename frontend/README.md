## YouTube Replica

### Step to run code

1. Fork the repository
2. Clone the repo or you can download zip file

```
cd file
npm install
npm run dev
```

### Features

1. Authentication & Authorization
2. Create Channel
3. Upload video
4. Comment on video
5. Line on video
6. Delete or edit your own comment
7. views channel
